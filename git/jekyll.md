# Jekyll

올해 목표 중 하나인 github에서 블로그를 운영하기 위해 Jekyll(지킬)을 설치한다. 구글 검색을 해보면 좋은 설치 자료도 많지만 읽을만한 구축 후기들이 많았다. 나도 깃헙 블로그를 구축하면서 앞으로 어떻게 운영할지 생각도 해보려고 한다. 2016년에는 왜 미루기만 했는지 반성도 필요하다.
